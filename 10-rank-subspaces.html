PART 3 · EQUATIONS & INVERSE

랭크와 부분공간 — 변환이 살려내는 차원

rank(랭크)는 행렬이라는 변환이 출력으로 살려내는 차원의 수예요. 2D를 그대로 평면으로 보내면 rank 2, 직선으로 짓이기면 rank 1, 원점 한 점으로 모으면 rank 0. 랭크는 곧 독립인 열의 개수이고, 열들이 만드는 공간이 열공간이에요. 슬라이더로 차원이 살고 죽는 걸 직접 보세요.

Step 00 — Rank = Surviving Dimension

rank = 변환이 살려내는 차원 수

행렬 네 칸을 직접 움직여 보세요. 출력이 평면 전체면 rank 2, 직선 하나로 납작해지면 rank 1, 원점 한 점으로 모이면 rank 0이에요. rank는 서로 독립인 열의 개수와 같아요.

행렬 M (열 = î, ĵ의 행선지)
1.0
0.5
-0.5
1.0

→ 두 열(î, ĵ)이 평행해지면(한 열이 다른 열의 배수) 출력이 직선으로 붕괴해 rank가 2에서 1로 떨어져요. 두 열이 모두 0이면 rank 0.

Step 01 — Rank 2 vs Rank 1

rank 2(평면 유지) vs rank 1(직선 붕괴)

두 행렬을 토글로 비교해요. 왼쪽 rank 2 행렬은 격자를 평면 그대로 보내지만, 오른쪽 rank 1 행렬은 두 열이 평행해서 격자 전체가 한 직선 위로 짜부라져요.

→ rank 1 행렬에서 두 열 (2,1)과 (4,2)는 한쪽이 다른 쪽의 2배예요. 같은 방향이라 새 차원을 못 만들어요.

Step 02 — Column Space = span of columns

열공간 = 열 벡터들의 span

행렬의 두 열을 벡터로 그렸어요. 이 두 벡터가 만드는 span(가능한 모든 조합)이 열공간이에요. 두 열이 다른 방향이면 span은 평면(음영 전체) = rank 2, 평행하면 직선 = rank 1. rank = 열공간의 차원이에요.

열 벡터 두 개
2.0
1.0
-1.0
1.5

→ 두 열을 평행하게 맞춰 보세요(예: 열2를 열1의 배수로). 음영이 평면에서 직선으로 줄며 열공간 차원이 2 → 1로 떨어져요.

Step 03 — Rank-Nullity

rank + nullity = 입력 차원

rank-nullity 정리: rank + nullity = 입력 차원(여기선 2). nullity는 변환이 0(원점)으로 보내버리는 입력들이 이루는 공간(영공간)의 차원이에요. 살아남는 차원(rank)과 으스러지는 차원(nullity)을 합치면 항상 입력 차원이 돼요.

ML 연결 — 저랭크 근사 큰 가중치 행렬을 rank를 일부러 낮춰 근사하면, 정보의 중요한 방향 몇 개만 남기고 나머지는 버려요. 파라미터가 확 줄죠. 딥러닝의 LoRA가 바로 이 아이디어예요 — 무거운 행렬을 두 개의 얇은(저랭크) 행렬 곱으로 대체해 핵심 방향만 학습시켜요.
