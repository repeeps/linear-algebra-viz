# 선형대수, 눈으로 보기 (Linear Algebra, Visualized)

벡터부터 SVD·PCA까지 — 선형대수의 핵심을 **공간이 휘는 격자**와 **실시간 계산**으로 직접 만지며 익히는 인터랙티브 학습 자료입니다. 행렬을 "공간을 변형하는 함수"로 보는 기하적 직관 위주로, 모든 변환·행렬식·고유값·분해가 브라우저에서 실제로 계산됩니다.

> 직접 만든 **비영리 학습용 자료**입니다. 특정 교재·강의와 무관하며, 설명·예시·코드는 모두 독자적으로 제작했습니다.

```bash

# 또는 간단한 로컬 서버
python3 -m http.server 8000     # → http://localhost:8000
```

## 챕터 (17개)

**PART 1 · 벡터와 공간**
| # | 주제 | 핵심 인터랙션 |
|---|------|--------------|
| 01 | 벡터란 무엇인가 | 화살표 ↔ 숫자목록 · 덧셈(꼬리물기) · 스칼라배 |
| 02 | 선형결합 · span · 기저 | a·v + b·w 로 평면 채우기 |
| 03 | 선형독립과 차원 | det 으로 종속 판정 · span 붕괴 |

**PART 2 · 선형변환과 행렬**
| # | 주제 | 핵심 인터랙션 |
|---|------|--------------|
| 04 | 선형변환 = 공간 변형 | 슬라이더로 격자가 휘는 모습 · î/ĵ |
| 05 | 행렬 = 변환의 좌표 | M·v = 열의 가중합 |
| 06 | 행렬곱 = 변환의 합성 | 합성 애니메이션 · AB ≠ BA |
| 07 | 행렬식 = 넓이 배율 | 단위정사각형 넓이 · 음수=뒤집힘 · 0=붕괴 |

**PART 3 · 연립방정식과 역**
| # | 주제 | 핵심 인터랙션 |
|---|------|--------------|
| 08 | 연립방정식 Ax = b | 두 직선의 교점 · 소거법 |
| 09 | 역행렬 · 열공간 · 영공간 | 변환 되돌리기 · 영공간 직선 |
| 10 | rank와 부분공간 | rank 붕괴 · 열공간 음영 |

**PART 4 · 내적과 직교**
| # | 주제 | 핵심 인터랙션 |
|---|------|--------------|
| 11 | 내적 = 투영 · 유사도 | 그림자 길이 · 부호 · 코사인 유사도 |
| 12 | 정사영과 최소제곱 | 점들에 가장 가까운 직선(회귀의 기하) |
| 13 | 직교기저 · Gram-Schmidt | 정사영 빼기로 직교화 |

**PART 5 · 고유값과 분해**
| # | 주제 | 핵심 인터랙션 |
|---|------|--------------|
| 14 | 고유값 · 고유벡터 | 방향이 안 바뀌는 벡터 찾기 |

| 16 | SVD — 모든 행렬의 분해 | 회전→스케일→회전 단계 애니메이션 |

## 설계 원칙

- **실제 계산**: 결과값 하드코딩 없이 JS가 매번 계산 (입력 샘플 데이터만 고정)
- **격자 변형 엔진**: 행렬을 점마다 실제로 적용해 격자가 휘는 모습을 그림 — "행렬 = 공간을 변형하는 함수"
- **단계별 분해**: 각 챕터 4스텝, 한 줄 직관 → 인터랙션 → 결론
